# Personal Expense Tracker (Python CLI)

A simple command-line application to track personal expenses.

## Features

- Add expenses with amount, category, description, and date
- List all recorded expenses
- View spending summary by category
- View spending summary by month
- Delete an expense by index
- Data stored locally in `expenses.json` (created automatically)

## Tech Stack

- Python 3
- Standard library: `json`, `datetime`, `os`, `typing`

## How to Run

```bash
python expense_tracker.py
